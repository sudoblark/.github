# Sudoblark

**Agentic AI · Platform Engineering · Full-Stack — Manchester, UK**

I'm Benjamin Clark. I build agentic AI systems, production-grade cloud platforms, and the data infrastructure to power them — end to end, no handoffs, no gaps.

---

## What I Build

**Agentic AI**
Autonomous agents using Amazon Bedrock and pydantic-ai: classification pipelines, RAG systems, semantic search, and natural language interfaces over structured data. Built to work in production, not just in demos.

**Platform Engineering**
Multi-account AWS with SSO, Terraform three-tier architecture, WAF-protected APIs, GitHub organisation management, and CI/CD with full quality and security gates. Infrastructure designed with proper abstractions — not ClickOps translated to HCL.

**Serverless Data Platforms**
Multi-tier data lakes (S3, Glue, Athena, DynamoDB), event-driven Step Functions pipelines, and Snowflake integration. OWASP controls built in. Documented to hand-off standard — your team inherits understanding, not dependency on me.

**Backend & Full-Stack**
Python APIs on ECS Fargate, Cognito-authenticated frontends, OpenAPI-first design, and automated test coverage. Documentation-first delivery: the systems I build are still running because the people maintaining them understand them.
